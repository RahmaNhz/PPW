Cosine similarity#

Cosine similarity adalah ukuran kesamaan antara dua vektor dalam ruang berdimensi tinggi. Nilai cosine similarity berkisar antara -1 dan 1, di mana 1 berarti vektor tersebut identik, 0 berarti tidak ada kesamaan, dan -1 berarti vektor tersebut berlawanan secara sempurna. Semakin kecil sudutnya, semakin besar nilai cosine similarity, yang berarti kedua vektor tersebut lebih mirip.

Rumus Cosine Similarity:

Jika kita memiliki dua vektor ( A ) dan ( B ), cosine similarity dapat dihitung dengan rumus berikut:

\[ \text{Cosine Similarity} = \cos(\theta) = \frac{A \cdot B}{|A| |B|} \]

Di mana:

\(( A \cdot B )\) adalah hasil perkalian dot \((dot product)\) dari vektor ( A ) dan ( B ). \(( |A| )\) adalah norma (panjang) dari vektor ( A ). \(( |B| )\) adalah norma (panjang) dari vektor ( B ).

from sklearn.metrics.pairwise import cosine_similarity

# Menghitung cosine similarity berdasarkan matriks TF-IDF dari kolom 'isi'
cosine_sim_matrix = cosine_similarity(tfidf_matrix)

Graf#

Graf adalah sebuah himpunan objek-objek yang dinamakan titik, simpul, atau sudut yang dihubungkan oleh penghubung yang dinamakan garis atau sisi. Graf digunakan untuk merepresentasikan hubungan antara objek-objek.

import pandas as pd
import networkx as nx
import matplotlib.pyplot as plt

# Membuat graf dari matriks biner
G = nx.from_pandas_adjacency(binary_cosine_sim_df)

# Menggambar graf
plt.figure(figsize=(12, 12))  # Menentukan ukuran figure
G.remove_edges_from([(u, u) for u in G.nodes()])
pos = nx.random_layout(G) # Layout untuk visualisasi graf
nx.draw(G, pos, with_labels=True, node_color='lightblue', node_size=700,
        font_size=15, font_color='black', edge_color='gray')



# Menampilkan graf
plt.title("Graf Keterhubungan Berita Berdasarkan Cosine Similarity")
plt.show()
