Part 3. PENERAPAN MODEL LOGISTIC REGRESSION UNTUK KLASIFIKASI BERITA#

Logistic Regression merupakan metode dalam pembelajaran mesin yang termasuk dalam supervised learning dan digunakan untuk memprediksi probabilitas hasil biner 1/0 (true/false) berdasarkan satu atau beberapa variabel fitur, degan menggunakan fungsi logistic (sigmoid) untuk melakukan prediksi.

Rumus Fungsi Sigmoid: $\( y = f(x) = \frac{1}{1 + e^{-z}} \)$

Di mana ( z ) adalah kombinasi linier dari variabel input (prediktor):

\[ z = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n \]

Maka:

\[ \hat{y} = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n)}} \]
